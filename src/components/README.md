# Components(组件)

组件属性及事件可参考[组件 · 小程序](https://mp.weixin.qq.com/debug/wxadoc/dev/component/)

## 示例

icon组件使用

```html
  <icon :type="type" :color="color" :size="size"></icon>
```

```javascript
  data () {
    type: 'success',
    color: 'green',
    size: 50
  }

```
